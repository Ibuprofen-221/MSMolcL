# MsMolCl
欢迎使用MsMolCl：基于对比学习的分子-质谱对齐模型网站
MsmolCl的核心思想是：采用对比学习的方法，将分子和质谱数据映射到同一个空间中，从而实现分子-质谱对齐。
# Online Chemical Library Search Tool
## 你可以做什么
本网站是围绕MsMolCl模型搭建的在线化学库检索工具，在这里你可以使用封装好的服务使用MsMolCl模型，实现数据库检索、二次检索、分子可视化、谱图分析对比等操作。
## task id
本网站使用"task id"工作流的方式来管理用户的上传文件和检索历史，您可以通过"普通检索"上传来获得您本次执行任务的独特id，为后续使用更多的功能做准备。
## 完整工作流
普通检索
—— 上传文件 得到task_id
       ——  选择候选池/离子模式
       ——  执行检索，更新历史记录
高级检索
—— 输入普通检索得到的task_id
       ——  选择离子模式
       ——  执行检索，更新历史记录
历史记录
—— 找到对应的task_id，进入"detail"，得到完整的可视化内容
## Notice
1. 您会发现，普通检索上传完成一次文件之后便不再允许上传。如果文件传输错误，需要手动执行clear操作
2. 普通检索的候选池/离子模式您可以进行多次更改，并进行多次检索。但只有最新的一次检索会被记录并且当做"高级检索"的依据
3. Pubchem候选池模式涉及到较大规模的分子检索过程，因此耗时较长

# 数据库检索
## 上传文件
· 质谱文件：mgf/txt格式
· 碎裂树文件：json格式 —— 碎裂树文件由sirius生成，您需要自行准备
MsMolCl支持同时预测不同加合物类型的谱图，但是要在质谱文件的MSLEVEL=2的块中添加 ADDUCTIONNAME=XXX一行，MsMolCl会依此自动识别加合物类型。
但需要注意的是：同一批上传的谱图文件中，加合物类型必须同时是正离子/负离子
## 离子模式
根据加合物类型的正负离子模式进行选择
即使您错误地选择了离子模式，模型依然会给出结果，但是不能保证正确性
## 选择数据库：
1. pubchem + n ppm检索。 MsMolCl会根据每个谱图的前体离子质量选择出其在pubchem中n ppm范围内的所有分子作为候选池。因需要对每个谱图分别进行检索，因此该模式下耗时最长
2. common检索。 MsMolCl设置了一些常用的分子作为候选池
3. costom检索。 您可以上传自己的文件作为候选池。文件要求为txt，smiles按行存放
## 结果返回：
MsMolCl会在候选池中排序出和谱图相似度最好的top10 smiles并展示。您可以通过"导出"下载结果文件，里边包含了更详细的检索信息。于此同时，这个文件还会用来当做"高级检索"的输入
## smiles可视化
您可以通过搜索框输入谱图的title，MsMolCl会使用rdkit工具将其top10 smiles可视化
# 高级检索：
MsMolCl通过对"普通检索"的top100结果进行重新训练并检索，能够大大提升对精细结构的区分能力，并在实验中得到了验证。因此，您可以完全信任高级检索得到的内容，将"普通检索"的结果作为参考
