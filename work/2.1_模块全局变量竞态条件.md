# 2.1 模块全局变量竞态条件 — 解决方案

## 问题定位

`backend/services/file_preprocess.py` 使用模块级全局变量存储输出路径，通过 `set_output_base()` 修改：

```python
# backend/services/file_preprocess.py:20-25
SPECTRUM_OUTPUT_DIR = str(_DEFAULT_TEMP_DIR)
FRAGTREE_OUTPUT_DIR = str(_DEFAULT_TEMP_DIR)
STATS_OUTPUT_PATH = str(_DEFAULT_TEMP_DIR / "statas.json")
VALID_PAIRS_MGF = str(_DEFAULT_TEMP_DIR / "valid_pairs_spectra.mgf")
VALID_PAIRS_JSON = str(_DEFAULT_TEMP_DIR / "valid_pairs_fragtrees.json")

# backend/services/file_preprocess.py:26-36
def set_output_base(base_dir: str | Path) -> None:
    """基于指定目录重置所有输出路径，确保任务隔离。"""
    global SPECTRUM_OUTPUT_DIR, FRAGTREE_OUTPUT_DIR, STATS_OUTPUT_PATH, VALID_PAIRS_MGF, VALID_PAIRS_JSON
    base_path = Path(base_dir)
    base_path.mkdir(parents=True, exist_ok=True)
    SPECTRUM_OUTPUT_DIR = str(base_path)
    # ... 修改其余全局变量
```

这些全局变量在以下位置被使用：

```python
# SpectrumNormalizer.__init__ (line 96)
Path(SPECTRUM_OUTPUT_DIR).mkdir(parents=True, exist_ok=True)

# SpectrumNormalizer.export_valid_pairs_mgf (line 343)
output_path = Path(VALID_PAIRS_MGF)

# FragTreeParser.__init__ (line 387)
Path(FRAGTREE_OUTPUT_DIR).mkdir(parents=True, exist_ok=True)

# FragTreeParser.export_valid_pairs_json (line 440)
output_path = Path(VALID_PAIRS_JSON)

# SpectrumFragTreeMatcher.match (line 509)
stats_path = Path(STATS_OUTPUT_PATH)
```

两个并发请求调用 `preprocess_main()` 时：
1. 请求 A 调用 `set_output_base("/user_A/task_001/pair_01")`
2. 请求 B 调用 `set_output_base("/user_B/task_002/pair_01")`
3. 请求 A 的 `SpectrumNormalizer` 读取到请求 B 设置的 `SPECTRUM_OUTPUT_DIR`
4. 数据写入错误目录

## 影响

- 用户 A 的数据可能写入用户 B 的 task 目录
- 输出文件可能部分缺失或内容错乱
- **严重程度: 高** — 并发上传场景确定可触发

## 解决方案

### 核心思路

**将全局路径变量改为实例属性**。`SpectrumNormalizer`、`FragTreeParser`、`SpectrumFragTreeMatcher` 均接收 `output_base_dir` 作为构造参数，内部计算所有输出路径。

### Step 1: 修改 `SpectrumNormalizer`

```python
# backend/services/file_preprocess.py（修改 SpectrumNormalizer）

class SpectrumNormalizer:
    def __init__(self, output_base_dir: str | Path):
        base = Path(output_base_dir)
        base.mkdir(parents=True, exist_ok=True)

        self.output_dir = str(base)
        self.statas_path = str(base / "statas.json")
        self.valid_pairs_mgf = str(base / "valid_pairs_spectra.mgf")
        self.valid_pairs_json = str(base / "valid_pairs_fragtrees.json")

        self.all_spectra: Dict[str, Dict[str, Any]] = {}
        self.raw_precision_cache: Dict[str, Dict[str, Any]] = {}
```

### Step 2: 修改 `FragTreeParser`

```python
# backend/services/file_preprocess.py（修改 FragTreeParser）

class FragTreeParser:
    def __init__(self, output_base_dir: str | Path):
        base = Path(output_base_dir)
        base.mkdir(parents=True, exist_ok=True)

        self.output_dir = str(base)
        self.statas_path = str(base / "statas.json")
        self.valid_pairs_mgf = str(base / "valid_pairs_spectra.mgf")
        self.valid_pairs_json = str(base / "valid_pairs_fragtrees.json")

        self.all_fragtrees: Dict[str, Any] = {}
```

### Step 3: 更新内部方法引用

在 `export_valid_pairs_mgf` 中使用 `self.valid_pairs_mgf` 替代全局 `VALID_PAIRS_MGF`：

```python
# 修改前
output_path = Path(VALID_PAIRS_MGF)

# 修改后
output_path = Path(self.valid_pairs_mgf)
```

同样替换：
- `SPECTRUM_OUTPUT_DIR` → `self.output_dir`
- `FRAGTREE_OUTPUT_DIR` → `self.output_dir`
- `STATS_OUTPUT_PATH` → `self.statas_path`
- `VALID_PAIRS_JSON` → `self.valid_pairs_json`

### Step 4: 修改 `SpectrumFragTreeMatcher.match`

将 `statas_path` 从全局变量改为由调用方传入：

```python
class SpectrumFragTreeMatcher:
    @staticmethod
    def match(
        spectrum_stats: Dict[str, Any],
        fragtree_stats: Dict[str, Any],
        spectrum_normalizer: SpectrumNormalizer,
        fragtree_parser: FragTreeParser,
    ) -> Dict[str, Any]:
        # ...

        # 修改前
        # stats_path = Path(STATS_OUTPUT_PATH)

        # 修改后（使用 instance 的路径）
        stats_path = Path(spectrum_normalizer.statas_path)
        with open(stats_path, 'w', encoding='utf-8') as f:
            json.dump(combined_stats, f, ensure_ascii=False, indent=2)

        return combined_stats
```

### Step 5: 修改 `main()` 函数

```python
# backend/services/file_preprocess.py（修改 main）

def main(
    spectrum_file_path: str = EXAMPLE_SPECTRUM_FILE,
    fragtree_file_path: str = EXAMPLE_FRAGTREE_FILE,
    output_base_dir: str | Path | None = None,
):
    if output_base_dir is None:
        output_base_dir = Path(spectrum_file_path).parent

    # 1. 创建带路径隔离的实例
    spectrum_normalizer = SpectrumNormalizer(output_base_dir=output_base_dir)
    _, spectrum_stats = spectrum_normalizer.parse_spectrum_file(str(spectrum_file_path))

    # 2. 创建带路径隔离的实例
    fragtree_parser = FragTreeParser(output_base_dir=output_base_dir)
    _, fragtree_stats = fragtree_parser.parse_fragtree_file(str(fragtree_file_path))

    # 3. 匹配并导出
    matcher = SpectrumFragTreeMatcher()
    combined_stats = matcher.match(
        spectrum_stats, fragtree_stats,
        spectrum_normalizer, fragtree_parser,
    )
    return combined_stats
```

### Step 6: 移除全局变量和 `set_output_base`

```python
# 删除以下代码:
# - SPECTRUM_OUTPUT_DIR
# - FRAGTREE_OUTPUT_DIR
# - STATS_OUTPUT_PATH
# - VALID_PAIRS_MGF
# - VALID_PAIRS_JSON
# - set_output_base() 函数
```

移除后，所有调用点（`file_upload.py`、`sirius_batch_service.py`）不再需要调用 `set_output_base()`。

### Step 7: 更新 `process_batch_pairs`

```python
# backend/services/file_preprocess.py（修改 process_batch_pairs）

def process_batch_pairs(
    pair_inputs: list[dict],
    output_base_dir: str | Path,
) -> dict:
    base_dir = Path(output_base_dir)
    base_dir.mkdir(parents=True, exist_ok=True)

    merged_root_info: list[dict] = []
    merged_valid_pairs: list[str] = []
    merged_fragtrees: dict = {}
    merged_spectra_text_parts: list[str] = []
    batch_items: list[dict] = []

    for idx, item in enumerate(pair_inputs, start=1):
        pair_key = str(item.get("pair_key") or f"pair_{idx}")
        spectrum_file = str(item.get("spectrum_file_path") or "")
        fragtree_file = str(item.get("fragtree_file_path") or "")
        if not spectrum_file or not fragtree_file:
            raise ValueError(f"批次项缺少输入文件: {pair_key}")

        pair_dir = base_dir / f"pair_{idx:03d}_{pair_key}"
        pair_dir.mkdir(parents=True, exist_ok=True)

        # main() 内部不再调用 set_output_base，改为传入 output_base_dir
        main(
            spectrum_file_path=spectrum_file,
            fragtree_file_path=fragtree_file,
            output_base_dir=pair_dir,
        )
        # ... 后续聚合逻辑不变
```
